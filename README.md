# user-repository
First we need to clone the git repo in our machine using this link- https://github.com/paulsatya/user-repository.git
Then take pull for any new updates.
To install dependancies, run "npm i" command in vs code terminal, this will create node modules folder.
Then build the app using build command- "ng build" or see command from package.json file.
To run the application locally we need to run - "npm start" command.
check the default browser as after successful compilcation, app will open automatically in browser.
